HISTPOL

Aplicação independente para coleta de relatos, gravação real de áudio por envolvido, ditado de voz, minuta assistida por IA, banco local no aparelho, backup em JSON e estrutura pronta para empacotamento Android com WebView ou Capacitor.

🎙️ gravação real por envolvido 📎 anexos por ocorrência e por envolvido 🗂️ banco local IndexedDB 🧾 backup e restauração JSON 📱 pronto para Android

Ocorrência e envolvidos

Preencha os dados básicos e adicione quantos envolvidos quiser, incluindo mais de uma vítima, mais de um condutor e múltiplos relatos policiais.

● banco local pronto ● modo app habilitável ● microfone aguardando uso ● anexos prontos
Use esta área para fotos, documentos, vídeos curtos, prints ou outros arquivos gerais da ocorrência.

A chave não fica embutida no arquivo. O app funciona sem chave para cadastro, áudio, anexos, backup e PDF. A chave só é necessária se você quiser usar a geração de minuta com IA ou a transcrição assistida de áudio. Se uma chave for exposta publicamente, ela pode ser bloqueada pelo Google.

Sem chave ativa. Os recursos locais continuam funcionando normalmente.

Adicionar envolvidos

Você pode tocar várias vezes no mesmo botão para criar Vítima 1, Vítima 2, Condutor 1, Condutor 2, Relato Policial 1, Relato Policial 2 e assim por diante.

Minuta, backup e exportação

Gere a minuta com IA, revise manualmente, exporte PDF, faça backup JSON com os áudios incorporados e importe tudo novamente quando precisar.

Dica: o backup exporta os campos da ocorrência, todos os envolvidos, os relatos, a minuta, os áudios gravados/importados e os anexos gerais ou por envolvido em formato incorporado no JSON.

Pronto para Android
Esta versão foi montada para rodar bem como HTML local e também pode ser empacotada em app Android. No pacote entregue junto, ela já vai com estrutura para Capacitor/WebView, manifest e service worker.
